ObjFile

The ObjFile class, which extends the Shape class. allows you to create Lumarca visuals with Wavefront .obj files. a standard 3D model file format. There are several options for creating, moving, rotating, resizing, displaying wireframes, and rendering these objects.

ObjFile class Constructor:

    ObjFile(PApplet pApplet, PVector center, PVector color, Sting color, float scale)
Important Fields:
    PVector center - the center of the shape
Functions
    rotateX(float f)
    rotateY(float f)
    rotateZ(float f)
LumarcaLibrary class options for ObjFile
    lumarca.drawShape(Shape file)
    lumarca.drawWireframe(Shape file)
