LumarcaLibrary class: getLines()

Example:

   void setup(){
   
   	Lumarca lumarca = new LumarcaLibrary(this, 256, false);  //create LumarcaLibrary

   	Line[] lines = lumarca.getLines();                   //Get all Lumarca Lines
              
   	for (int lineNum = 0; lineNum < lines.length; lineNum++) { //Go through all the lines
           	Line line = lines[lineNum];                        //Get each line                   
            	println(line.bottom);							   //Print the bottom coord
   	}
	}

Description

    Returns an aray of all the lines in the lumarca.
    This is useful for renderings based on the position of individual lines, such as math equations.

Parameteres

    None.
