LumarcaLibrary class: LumarcaLibrary(PApplet pApplet, String fileName)

Example:

   void setup(){
   
   	Lumarca lumarca = new LumarcaLibrary(this, "lineConfig256.txt");  //create LumarcaLibrary
	
		PVector minPos = new PVector();
		lumarca.getMinPosition(minPos);

   	println(minPos);        //print out the x,y,z value of the edge of the volume
	}

Description

    Constructor. Create a LumarcaLibraryobject used to make lumarca content.

Parameteres

    PApplet
      Gives the Lumarca Library access to your sketch and all of the base processing functionality.
    String
      Filename of a pre-generated configuration for a Lumarca. Useful if you are running your content on a Lumarca that has already been configured to that calibration pattern.
