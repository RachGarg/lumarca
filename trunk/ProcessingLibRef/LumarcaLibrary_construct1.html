Description

    Constructor. Create a LumarcaLibraryobject used to make lumarca content.

Parameteres

    PApplet
      Gives the Lumarca Library access to your sketch and all of the base processing functionality.
    int
      Number of lines in this lumarca configurations (example: 64, 128, 256, etc.)
    boolean
      Determines whether to use a previously constructed configuration of lines or a new one. If it is false, it will create a new configuration, if true it will use a previously formed configuration.
