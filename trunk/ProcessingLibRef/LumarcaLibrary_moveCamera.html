LumarcaLibrary class: moveCamera(float rot)

Example:

  
  Shape shape;
  float rot = 0;
  
   void setup(){
   		Lumarca lumarca = new LumarcaLibrary(this, 256, false);  //create LumarcaLibrary
        
        	shape = new ObjFile(this, center, new PVector(1, 0, 0), "obj/cube.obj", 300);
	}
    
    void draw() {
			rot += 0.1;
           lumarca.drawWireFrame(shape);
			lumarca.moveCamera(rot);  
     }

Description

    Rotate the camera out of projection mode, into simulated 3D mode.
    This is useful when attempting to simulate an animation so you can get an idea of what it will look like on the strings.
    Note: Will not work projected on a Lumarca, for testing and simulation only.

Parameteres

    float rot
      amount to rotate camera

Related
