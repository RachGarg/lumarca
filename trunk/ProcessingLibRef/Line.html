Line

The Line class holds the top and bottom coordinates for a given line.


Line class

Important Fields:

    PVector bottom - the bottom of the line.
    PVector top - the top of the line.

Example usage: 

   Line[] lines = lumarca.getLines();                   //Get all Lumarca Lines
              
	for (int lineNum = 0; lineNum < lines.length; lineNum++) { //Go through all the lines
		Line line = lines[lineNum];                        //Get each line                   
		println(line.bottom);							   //Print the bottom coord        
		println(line.top);							   	   //Print the top coord
	}
