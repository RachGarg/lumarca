LumarcaLibrary class: getCenterPosition(PVector p)

Example:

   void setup(){
   
   	Lumarca lumarca = new LumarcaLibrary(this, 256, false);  //create LumarcaLibrary
	
		PVector center = new PVector();
		lumarca.getCenterPosition(center);

   	println(center);        //print out the x,y,z value of the center of the volume
	}

Description

    Sets the paased PVector to the middle coordinates of the volume.
    This is useful for placing shapes in the center of the volume.

    Note: this function does not return anything. It accepts a PVector and sets it value. This is useful to reduce the number of objects produced and a standard approach when dealing with graphics programming.

Parameteres

    PVector
      This PVector is set to have the x,y,z coordinates of the middle of the volume.
