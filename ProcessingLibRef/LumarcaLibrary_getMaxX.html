LumarcaLibrary class: getMaxX()

Example:

   void setup(){
   
   	Lumarca lumarca = new LumarcaLibrary(this, 256, false);  //create LumarcaLibrary

   	println(lumarca.getMaxX());        //print out maximum X position
	}

Description

    Returns the maximum X value of any line the volume.
    This is useful for finding the bounds of the volume, for example, if you have a shape moving through the volume and you don't want it move beyond the edge.

Parameteres

    None.
