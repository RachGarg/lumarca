LumarcaLibrary class: getMidZ()

Example:

   void setup(){
   
   	Lumarca lumarca = new LumarcaLibrary(this, 256, false);  //create LumarcaLibrary

   	println(lumarca.getMidZ());        //print out middle Z position
	}

Description

    Returns the middle Z value of the volume.
    This is useful for placing shapes in the center of the volume.

Parameteres

    None.
