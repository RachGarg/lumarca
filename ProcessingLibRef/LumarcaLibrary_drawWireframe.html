LumarcaLibrary class: drawWireFrame(Shape shape)

Example:

  
  Shape shape;
  
   void setup(){
   		Lumarca lumarca = new LumarcaLibrary(this, 256, false);  //create LumarcaLibrary
        
        	shape = new ObjFile(this, center, new PVector(1, 0, 0), "obj/cube.obj", 300);
	}
    
    void draw() {
           lumarca.drawWireFrame(shape);  
     }

Description

    Draw the wireframe of the shape that was passed.
    This is useful for seeing where an object is in 3D space and what scale it is.
    Note: Will not work projected on a Lumarca, for testing and simulation only.

Parameteres

    Shape shape
      The Shape to be drawn.
