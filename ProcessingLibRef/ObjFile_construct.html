ObjFile class: ObjFile(PApplet pApplet, PVector center, PVector color, Sting filename, float scale)

Example:

  Shape shape;
  
   void setup(){
   	Lumarca lumarca = new LumarcaLibrary(this, 256, false);  //create LumarcaLibrary
        
       PVector center = new PVector();
		lumarca.getCenterPosition(center);
       shape = new ObjFile(this, center, new PVector(1, 0, 0), "obj/cube.obj", 300);
	}
    
	void draw() {
       lumarca.drawShape(shape.color, shape);  
   }

Description

    Constructor. Create a ObjFile that can be rendered inside the Lumarca.

Parameteres

    PApplet
      Gives the ObjFile Class access to your sketch and all of the base processing functionality.
    PVector center
      Center position for the object
    PVector color
      This PVector sets the color of the line. The x is the red value, the y is the green value, and the blue is the y value.
      Note: Lumarca colors go from 0-1, not 0-255. So if you want something to be half red, the value is 0.5.
    String filename
      name of the obj file you with to load
    float scale
      The size at which you wish to render the OBJ file. Note: Obj files vary greatly in size. You may want to draw it with the "drawWireframe" function of the Lumarca Library to see how much this has to be adjusted for your Obj Files.
