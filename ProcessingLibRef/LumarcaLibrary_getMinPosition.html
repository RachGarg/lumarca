Description

    Sets the passed PVector to the minimum coordinates of the volume.
    This is useful for placing shapes at the edge of the volume or bounding motion inside of the volume.

    Note: this function does not return anything. It accepts a PVector and sets it value. This is useful to reduce the number of objects produced and a standard approach when dealing with graphics programming.

Parameteres

    PVector
      This PVector is set to have the x,y,z coordinates of the minimum of the volume.
